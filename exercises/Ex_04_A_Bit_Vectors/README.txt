Note that, compared to previous exercises, the Makefile uses more
bsc flags to organize source files, intermediate files, etc.

When you use any of the 'make' commands, it shows the 'bsc' command
that is performed.  Study the 'bsc' commands and make sure you
understand all the command-line flags.

    bsc -help

shows a one-line help for each command-line arg.
The 'BSC Users Guide' provides more details.
